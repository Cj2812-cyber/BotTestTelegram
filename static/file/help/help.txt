Базовые команды:
/help - справка, список команд
/start - запустить бота и приветствие
/cancel - отмена выполнения текущего сценария
/drafts - получить черновик заявления по услуге ЕПГУ 2 из Cassandra. Необходимо указать номера заявлений, каждый номер на новой строке
/get_file_s3 - скачать и сохранить файл с s3. Необходимо указать BUCKET + REAL_PATH для файла из БД (например: epgu202102/51/0/0/24/24/65/3/ljCP2P0lS4b9)
/logs - возвращает файлы логов ЕПГУ
